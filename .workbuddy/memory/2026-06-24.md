## CDN 资源全面本地化

### 问题
jsdelivr.net / cdn.datatables.net / unpkg.com 等 CDN 在内网环境完全不可达（ERR_CONNECTION_CLOSED），导致 Bootstrap/jQuery/Chart.js/DataTables 全部加载失败，页面样式崩溃、Chart is not defined 等错误。

### 解决方案
将所有 CDN 依赖下载到 `app/static/vendor/` 目录，模板引用改为 `url_for('static', filename='vendor/...')`。

### 下载来源
- **bootcdn.net**（国内可用）：Bootstrap 5.3.2 CSS/JS、Bootstrap Icons 1.11.2 CSS+字体(woff/woff2)、jQuery 3.7.1、Chart.js 4.4.0
- **npm 镜像**（registry.npmmirror.com）：DataTables 1.13.8（jquery.dataTables.min.js / dataTables.bootstrap5.min.js/.css）、DataTables Buttons 2.4.2（buttons.bootstrap5.min.js/.css / dataTables.buttons.min.js）
- **自建**：datatables-zh.json（中文语言包）
