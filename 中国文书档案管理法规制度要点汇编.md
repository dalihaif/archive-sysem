# 中国文书档案管理法规制度要点汇编

> 整理日期：2026年6月15日

---

## 一、《中华人民共和国档案法》核心要点

### 1. 基本信息
- **最新修订**：2020年6月20日修订，自2021年1月1日起施行
- **实施条例**：2024年1月12日公布（国务院令第772号），2024年3月1日起施行，废止原《档案法实施办法》

### 2. 立法目的
加强档案管理，规范档案收集、整理工作，有效保护和利用档案，提高档案信息化建设水平，推进国家治理体系和治理能力现代化。

### 3. 核心管理原则

| 原则 | 具体要求 |
|------|----------|
| **党的领导** | 坚持和加强党对档案工作的领导，贯穿档案工作各方面和各环节 |
| **第一责任人制度** | 单位主要负责人承担档案完整与安全第一责任人职责 |
| **集中统一管理** | 任何内设机构和个人不得拒绝归档或据为己有 |
| **分级管理** | 国家档案馆馆藏永久保管档案分为一、二、三级 |

### 4. 档案移交期限

| 档案馆级别 | 移交期限 |
|-----------|---------|
| 中央级、省级、设区的市级 | 自档案形成之日起满**20年** |
| 县级 | 自档案形成之日起满**10年** |

- 专业性较强或需保密的档案，经批准可**延长**移交期限
- 保管条件不符合要求的可**提前**移交

### 5. 档案保管要求
- 建立科学管理制度和安全风险管控措施、应急预案
- 配置专门库房，配备防火、防盗、防水、防光、防尘、防有害气体、防有害生物及温湿度调控等设施
- 编制检索工具便于查找利用

### 6. 档案出境管理
- **一级档案严禁出境**
- **二级档案**出境须经国家档案主管部门审查批准
- 其他档案须按管理权限报批，海关凭批准文件查验放行
- 涉及数据出境的，须符合国家关于数据出境的规定

### 7. 档案信息化建设要求
- 将档案信息化**纳入本单位信息化建设规划**
- 加强办公自动化系统、业务系统**归档功能**建设
- 电子档案须保证**来源可靠、程序规范、要素合规**
- 档案馆接收时须进行**真实性、完整性、可用性和安全性**检测
- 重要电子档案应采用多种存储介质**异地备份**
- 推进数字档案馆（室）建设

### 8. 档案开放利用
- 国家档案馆应**分期分批向社会开放档案**
- 建立**馆藏档案开放审核协同机制**
- 提供阅览、复制、摘录等形式利用
- 载有签章标识的**复制件与原件具有同等效力**

### 9. 法律责任
- 擅自扩大或缩小接收范围、不按规定开放利用 → 责令限期改正；情节严重的依法给予处分
- 拒绝归档、不按规定移交档案 → 责令限期改正；拒不改正的依法给予处分
- 侵占、挪用国家档案馆馆舍 → 构成犯罪的追究刑事责任

## 二、《机关文件材料归档范围和文书档案保管期限规定》（国家档案局令第8号）

### 1. 基本信息
- **发布日期**：2006年12月18日，自公布之日起施行
- **废止**：1987年《国家档案局关于机关档案保管期限的规定》和《机关文件材料归档和不归档的范围》
- **适用范围**：各级党政机关和人民团体；军队系统、民主党派、企业事业单位参照执行

### 2. 归档范围（四类）

| 序号 | 归档类型 |
|------|----------|
| 1 | 反映本机关主要职能活动和基本历史面貌的，对本机关工作、国家建设和历史研究具有利用价值的文件材料 |
| 2 | 机关工作活动中形成的在维护国家、集体和公民权益等方面具有凭证价值的文件材料 |
| 3 | 本机关需要贯彻执行的上级机关、同级机关的文件材料；下级机关报送的重要文件材料 |
| 4 | 其他对本机关工作具有查考价值的文件材料 |

### 3. 不归档范围（四类）

| 来源 | 不归档情形 |
|------|-----------|
| **上级机关** | 普发性不需本机关办理的文件；任免、奖惩非本机关工作人员的文件；供工作参考的抄件 |
| **本机关** | 重份文件；无查考利用价值的事务性、临时性文件；一般性文件的历次修改稿、各次校对稿；无特殊保存价值的信封；不需办理的一般性人民来信、电话记录；机关内部互相抄送的文件；负责人兼任外单位职务形成的与本机关无关的文件 |
| **同级机关** | 不需贯彻执行的文件；不需办理的抄送文件 |
| **下级机关** | 供参阅的简报、情况反映；抄报或越级抄报的文件 |

### 4. 归档基本要求
- 实行**集中统一管理**，任何个人不得据为己有或拒绝归档
- **电子文件**应归档元数据、背景信息等
- 有文件发文稿纸、文件处理单的，应与文件**正本、定稿一并归档**
- 联合行文**原件由主办机关归档**，其他机关归档复制件
- 各机关须编制本机关归档范围和保管期限表，经**同级档案行政管理部门审查同意**后执行

## 三、《归档文件整理规则》（DA/T 22-2015）核心要求

### 1. 基本信息
- 代替DA/T 22-2000，推荐性行业标准
- 适用范围扩展为**纸质和电子文件材料**

### 2. 整理原则
1. **规律性与联系性**：遵循文件的形成规律，保持文件之间的有机联系
2. **区分价值**：区分不同价值，便于保管和利用
3. **文档一体化**：符合文档一体化管理要求，便于计算机管理
4. **纸电协调**：保证纸质文件和电子文件整理协调统一

### 3. 件的组织（组件）
- 一般以**每份文件为一件**
- 正文与附件、正本与定稿、转发文与被转发文各为一件
- 来文与复文一般独立成件，也可合为一件
- 有文件处理单的，处理单与相关文件为一件

**件内排序规则**：
- 正文在前，附件在后；正本在前，定稿在后
- 转发文在前，被转发文在后；原件在前，复制件在后
- 来文与复文合为一件时，**复文在前，来文在后**
- 文件处理单在前，收文在后

### 4. 分类方法

**三级分类（一般单位）**：
- 年度—机构（问题）—保管期限
- 年度—保管期限—机构（问题）

> 机构分类法与问题分类法**不可同时采用**

**两级分类（规模较小单位）**：
- 年度—保管期限

**年度判定规则**：
- 跨年度文件以**签发日期**为准
- 计划/总结/预算按**签发日期**判定
- 会议文件归入**闭幕年**
- 跨年度办理文件归入**办结年**

### 5. 档号编制

**档号结构**：

```
全宗号 - 档案门类代码·年度 - 保管期限 - 机构（问题）代码 - 件号
```

| 项目 | 编制规则 |
|------|----------|
| 全宗号 | 4位数字或字母与数字结合 |
| 档案门类代码 | 文书档案为**"WS"** |
| 年度 | 4位阿拉伯数字（如2013） |
| 保管期限 | 永久=**Y**，30年=**D30**，10年=**D10** |
| 机构代码 | 3位汉语拼音字母（如办公室="BGS"） |
| 件号 | 4位阿拉伯数字，不足前补"0" |

**示例**：`Z109-WS·2011-Y-BGS-0001`

### 6. 装订要求

| 保管期限 | 装订方式 |
|----------|----------|
| **永久** | 宜采取线装法（直角装订或"三孔一线"），**禁用**不锈钢夹或封套 |
| **定期（需移交进馆）** | 按永久保管要求执行 |
| **定期（不需移交进馆）** | 可使用不锈钢夹或封套装订 |

**禁用材料**：回形针、大头针、燕尾夹、热熔胶、办公胶水、装订夹条、塑料封

### 7. 编目与排架
- 目录项目：**序号、档号、文号、责任者、题名、日期、密级、页数、备注**
- 题名不规范或不反映主要内容的，应自拟标题加**方括号 [ ]**
- 不同年度、机构（问题）、保管期限的归档文件**不能装入同一个档案盒**
- 档案盒外形：**310mm x 220mm**，须采用**无酸纸**制作
- 排架方法应与分类方案一致，避免频繁倒架

## 四、《文书档案案卷格式》（GB/T 9705-2008）规范

### 1. 适用范围
按**立卷方法**整理档案的各级各类档案馆（室）和文书处理部门

> 注：DA/T 22-2015推行的是以"件"为单位的整理方法；GB/T 9705-2008规范的是以"卷"为单位的传统立卷方法，两者并存使用。

### 2. 硬卷皮规格
- **封面/封底尺寸**：310mm x 220mm
- **卷脊厚度**：10mm、15mm、20mm三种
- **推荐用材**：250g无酸牛皮纸

**封面项目（9项）**：全宗名称、类目名称、案卷题名、时间、保管期限、件/页数、归档号、档号

**卷脊项目**：全宗号、目录号、年度、案卷号、归档号

### 3. 软卷皮规格
- **封面/封底尺寸**：297mm x 210mm（A4）
- 软卷皮**必须装入卷盒内保存**

### 4. 卷盒规格
- **外形尺寸**：310mm x 220mm
- **厚度规格**：30mm、40mm、50mm

### 5. 卷内文件目录
- **幅面**：A4（297mm x 210mm）
- **页边距**：上20mm、下15mm、左25mm、右15mm（精度±0.5mm）

| 项目 | 填写要求 |
|------|----------|
| 顺序号 | 卷内文件排列先后顺次填写的序号 |
| 文号 | 文件制发机关的发文字号 |
| 责任者 | 团体用通用简称（禁"本局""本部"）；个人著录姓名；联合行文著录首位责任者 |
| 题名 | 照实抄录标题；无标题或标题不准确时自拟加**\[ \]** |
| 日期 | **8位数字**表示（YYYYMMDD） |
| 页号 | 首件首页对应页号；末件填写起止页号；从1开始编制通号 |

### 6. 案卷排列

**硬卷皮组卷**：案卷封面 → 卷内文件目录 → 文件 → 备考表 → 封底

**软卷皮组卷**：软卷封面（含卷内文件目录）→ 文件 → 封底（含备考表）→ 装入卷盒

### 7. 填写总体要求
- 使用**符合耐久性要求的字迹材料**
- 字迹工整清晰
- 各种项目位置、尺寸严格按照标准规定执行

## 五、档案分类编号规则（档号编制）

### 1. 主要依据标准
- **DA/T 13-2022《档号编制规则》**（代替DA/T 13-1994）

### 2. 档号编制原则

| 原则 | 说明 |
|------|------|
| **唯一性** | 同一档案馆内，一个档号只对应一份/卷档案实体 |
| **合理性** | 档号结构必须与档案的整理分类体系相一致 |
| **稳定性** | 档号一经确定，不应随意改动 |
| **简明性** | 档号长度和字符应尽量简明，避免冗长 |
| **扩容性** | 留有适当的递增空间，便于后续档案插入和延展 |

### 3. 档号基本结构

```
全宗号 - 档案门类代码·年度 - 保管期限 - 机构（问题）代码 - 件号
```

**符号规则**：
- 上、下位代码之间用 **"-"** 连接
- 同一级代码之间用 **"·"** 隔开

### 4. 常用档案门类代码

| 代码 | 档案门类 |
|------|----------|
| WS | 文书档案 |
| KY | 科技档案 |
| JJ | 会计档案 |
| JP | 基建档案 |
| ZP | 照片档案 |
| SW | 声像档案 |
| RZ | 人事档案 |
| DZ | 电子档案 |

### 5. 各著录层级档号结构（DA/T 18-2022）

| 层级 | 档号结构 | 示例 |
|------|---------|------|
| 全宗级 | 全宗号 | `X033` |
| 类别级 | 全宗号-类别号 | `X033-ZP` |
| 案卷级 | 全宗号-类别号-案卷号 | `X033-ZP·2008-001` |
| 文件级 | 全宗号-类别号-案卷号-件号 | `A002-WS·2019·Y-0001` |

- **馆际交流**：档案馆代码-档号
- **国际交流**：国家代码-档案馆代码-档号

## 六、文书档案保管期限划分标准

### 1. 期限种类

保管期限定为**两种**：
- **永久**
- **定期**：分为**30年**、**10年**

> 旧分类（2006年前）：永久、长期（16-50年）、短期（15年以下），已不再使用。

### 2. 永久保管标准

| 序号 | 永久保管范围 |
|------|-------------|
| 1 | 本机关制定的**法规政策性**文件材料 |
| 2 | 本机关召开**重要会议**、举办**重大活动**等形成的主要文件材料 |
| 3 | 本机关职能活动中形成的**重要业务**文件材料 |
| 4 | 本机关关于**重要问题**的请示与上级机关批复、批示，重要的报告、总结、综合统计报表等 |
| 5 | 本机关**机构演变、人事任免**等文件材料 |
| 6 | 本机关**房屋买卖、土地征用**，重要的合同协议、资产登记等凭证性文件材料 |
| 7 | 上级机关制发的属于本机关主管业务的**重要**文件材料 |
| 8 | 同级机关、下级机关关于**重要业务问题**的来函、请示与复函、批复等 |

**简要判断标准**：凡是反映机关主要职能活动和基本历史面貌的，对本机关、国家和社会具有**长远利用价值**的文件。

### 3. 定期30年保管标准

| 序号 | 30年保管范围 |
|------|-------------|
| 1 | 本机关职能活动中形成的**一般性业务**文件材料 |
| 2 | 本机关召开会议、举办活动等形成的**一般性**文件材料 |
| 3 | 本机关人事管理工作形成的**一般性**文件材料 |
| 4 | 本机关**一般性事务管理**文件材料 |
| 5 | 本机关关于**一般性问题**的请示与批复，一般性报告、总结、统计报表 |
| 6 | 上级机关制发的属于本机关主管业务的**一般性**文件材料 |
| 7 | 上级机关和同级机关制发的**非本机关主管业务但要贯彻执行**的文件材料 |
| 8 | 同级机关、下级机关关于**一般性业务问题**的来函、请示与复函、批复 |
| 9 | 下级机关报送的**年度或年度以上**计划、总结、统计、重要专题报告 |

**简要判断标准**：反映机关一般工作活动，在**较长时期**内具有查考利用价值。

### 4. 定期10年保管标准

| 序号 | 10年保管范围 |
|------|-------------|
| 1 | 供参阅的一般性上级文件、简报 |
| 2 | 机关内部一般性事务性文件 |
| 3 | 一般性问题的请示与批复 |
| 4 | 下级机关报送的年度以下计划、总结、统计等 |

**简要判断标准**：在**较短时间**内有查考利用价值的文件，如事务性、临时性文件。

### 5. 保管期限划分速查表（按文件类型）

| 文件类型 | 永久 | 30年 | 10年 |
|----------|------|------|------|
| 党代会/人代会文件 | 请示批复/决议/纪要 | 筹备工作/小组记录 | 未通过文件 |
| 党组/行政办公会纪要 | 全部 | — | — |
| 重要会议 | 请示批复/决议/纪要 | 典型材料/简报 | — |
| 上级检查视察 | 重要的 | 一般的、工作汇报 | — |
| 法规政策/中长期规划 | 全部 | — | — |
| 重要业务请示批复 | 重要的 | 一般的 | — |
| 行政处罚 | 重要的 | 一般的 | — |
| 年度以上计划总结 | 全部 | — | — |
| 年度以下计划总结 | — | — | 全部 |
| 机构设置/人事任免 | 全部 | — | — |
| 县以上表彰 | 全部 | — | — |
| 县以下表彰 | — | 全部 | — |

## 七、档案著录规则（DA/T 18-2022）

### 1. 基本信息
- 代替DA/T 18-1999
- 适用于各级各类档案馆（室）对各类档案的著录
- 涵盖传统载体档案和数字档案

### 2. 四项著录原则

| 原则 | 说明 |
|------|------|
| **层级性** | 从文件级到全宗级分级著录，反映部分与整体的层次结构 |
| **相称性** | 仅记录著录单元所在层级的相关信息 |
| **关联性** | 标识著录层级，与上一级著录单元连接 |
| **动态性** | 著录内容变化时应重新著录 |

### 3. 著录层级（四级）

```
全宗级 → 类别级 → 案卷级 → 文件级
```

### 4. 著录项目清单（7大类）

| 大类 | 包含项目 |
|------|----------|
| **标识项** | 档案馆代码、著录层级、档号、题名、文件编号、日期 |
| **背景项** | 责任者、组织机构沿革/人物生平、档案保管沿革 |
| **内容与结构项** | 范围和提要、人名、稿本、文种、附件、载体形态、计算机文件大小、计算机文件格式、生成方式、整理情况、保管期限、销毁情况 |
| **查阅与利用控制项** | 密级、公开属性、开放标识、语言或文字、主题词或关键词、分类号、缩微号、存储位置 |
| **相关档案材料项** | 原件存放位置、复制件存放位置、相关著录单元 |
| **附注项** | 各项目补充说明、考证依据等 |
| **著录控制项** | 著录者、著录日期 |

### 5. 关键著录格式

**题名著录**：
- 有题名照原文著录
- 题名冗长或不准确：原题名照录 + 另拟题名加 `[ ]`
- 无题名：依据内容拟写加 `[ ]`
- 并列题名前加 `=`，副题名前加 `:`

**日期著录**：
- 格式：**YYYYMMDD**（8位阿拉伯数字）
- 历史档案朝代纪年照录，附公元纪年
- 考证日期加 `[ ]`
- 不完整日期缺位用 `□` 补之

**责任者著录**：
- 多个责任者用 `;` 隔开
- 省略的用 `[等]`
- 别名/笔名照录，真实名称附后加 `( )`
- 考证责任者加 `[ ]`

### 6. 电子档案著录特殊要求

| 项目 | 要求 |
|------|------|
| 计算机文件大小 | 以MB为计量单位（如 0.14MB） |
| 计算机文件格式 | 记录格式名称，如 ofd、pdf、doc、jpg、mp4 等 |
| 生成方式 | 分三类：**原生**（电子档案）、**数字化**（数字副本）、**编辑**（经剪辑形成） |
| 存储位置 | 著录脱机载体类型、编号和存址 |

### 7. 保管期限著录
- 2006年前：永久/长期/短期
- 2006年后：永久/定期（30年）

### 8. 开放标识
- 开放/控制/延期开放/未审核
